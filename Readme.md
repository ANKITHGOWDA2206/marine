# 🌊 SeaGuardian

**SeaGuardian** is a powerful environmental monitoring platform designed to detect pollution through AI-powered image analysis and track e-waste in real time across India. It also raises awareness about environmental issues, endangered species, wildlife abuse, and supports active NGOs through donations and programs.

---

## 🔥 Live Site  
🔗 [https://seaguardian.binit.site](https://seaguardian.binit.site)

---

## 🎯 Project Highlights

### 📷 AI-Powered Pollution Detection  
- Analyze uploaded images to detect land, water, or urban pollution using AI.

### 🗺️ Real-Time E-Waste Tracker  
- Track and visualize city-wise e-waste dumping across India.
- Interactive and data-rich geospatial map interface.

### 📰 Real-Time Environmental News  
- Stay informed with live updates fetched from trusted environmental news sources.

### 🏢 NGO Programs & Partnerships  
- Discover NGOs working on pollution control and wildlife conservation.
- Explore programs and get involved in real change.

### 🐾 Endangered Species Section  
- Learn about India's most vulnerable species and what’s threatening their existence.

### 🚨 Wildlife Abuse Reports  
- View reports of inhumane acts affecting wildlife to raise awareness and action.

### 💸 Secure Donation System  
- Donate directly to NGOs and verified organizations through the platform.

---

## 💻 Tech Stack

- **Frontend**: React (Vite)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: Clerk (User Site), Firebase (Admin Panel)
- **Hosting**: Vercel (Serverless Architecture)
- **External APIs**:  
  - AI Model API for pollution detection  
  - News API for real-time articles
